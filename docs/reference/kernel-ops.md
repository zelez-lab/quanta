# KernelOp Reference

The 51 IR operations that represent GPU kernel logic. Generated by
`#[quanta::kernel]` at compile time. Consumed by backend code generators
(SPIR-V, metallib, PTX, GCN).

> This page covers the commonly-authored ops. The full enum
> (`crates/quanta-ir/src/types.rs`, `KernelOp`) also includes
> synchronization (`Fence`, `SharedAtomicOp`), quantization
> (`Quantize` / `Dequantize` — per-tensor symmetric int8/int4),
> cooperative matrix (`CooperativeMMA`), dynamic shared memory
> (`SharedDeclDyn`), and `DebugPrint`. `types.rs` is the source of truth.

---

## Memory

### `Load`
Read a value from a GPU buffer field.
```
Load { dst: Reg, field: u32, index: Reg, ty: ScalarType }
```
`dst = fields[field][index]` as type `ty`.

### `Store`
Write a value to a GPU buffer field.
```
Store { field: u32, index: Reg, src: Reg, ty: ScalarType }
```
`fields[field][index] = src` as type `ty`.

### `SharedDecl`
Declare a block of workgroup-local shared memory.
```
SharedDecl { id: u32, ty: ScalarType, count: u32 }
```
Allocates `count` elements of type `ty` in shared memory, identified by `id`.

### `SharedLoad`
Read a value from shared memory.
```
SharedLoad { dst: Reg, id: u32, index: Reg, ty: ScalarType }
```
`dst = shared[id][index]`.

### `SharedStore`
Write a value to shared memory.
```
SharedStore { id: u32, index: Reg, src: Reg, ty: ScalarType }
```
`shared[id][index] = src`.

## Arithmetic

### `BinOp`
Binary arithmetic or bitwise operation.
```
BinOp { dst: Reg, a: Reg, b: Reg, op: BinOp, ty: ScalarType }
```

| `op` | Operation |
|------|-----------|
| `Add` | `a + b` |
| `Sub` | `a - b` |
| `Mul` | `a * b` |
| `Div` | `a / b` |
| `Rem` | `a % b` |
| `BitAnd` | `a & b` |
| `BitOr` | `a \| b` |
| `BitXor` | `a ^ b` |
| `Shl` | `a << b` |
| `Shr` | `a >> b` |

### `UnaryOp`
Unary operation.
```
UnaryOp { dst: Reg, a: Reg, op: UnaryOp, ty: ScalarType }
```

| `op` | Operation |
|------|-----------|
| `Neg` | `-a` |
| `BitNot` | `!a` (bitwise) |
| `LogicalNot` | `!a` (logical) |

### `Cmp`
Comparison operation. Result is `Bool`.
```
Cmp { dst: Reg, a: Reg, b: Reg, op: CmpOp, ty: ScalarType }
```

| `op` | Operation |
|------|-----------|
| `Eq` | `a == b` |
| `Ne` | `a != b` |
| `Lt` | `a < b` |
| `Le` | `a <= b` |
| `Gt` | `a > b` |
| `Ge` | `a >= b` |

### `Cast`
Type conversion.
```
Cast { dst: Reg, src: Reg, from: ScalarType, to: ScalarType }
```

### `Bitcast`
Reinterpret the bits of a value as a different type (same bit width required).
```
Bitcast { dst: Reg, src: Reg, from: ScalarType, to: ScalarType }
```
`dst = bitcast<to>(src)`. No value conversion -- bits are preserved. Common use:
`f32` <-> `u32` for float bit manipulation (e.g., fast inverse square root).

### `Const`
Load a constant value into a register.
```
Const { dst: Reg, value: ConstValue }
```

`ConstValue` variants: `F16(u16)`, `BF16(u16)`, `FP8E5M2(u8)`, `FP8E4M3(u8)`, `F32(f32)`, `F64(f64)`, `U32(u32)`, `U64(u64)`, `I32(i32)`, `I64(i64)`, `Bool(bool)`.

### `CountTrailingZeros`
Count the number of trailing zero bits.
```
CountTrailingZeros { dst: Reg, src: Reg, ty: ScalarType }
```
`dst = ctz(src)`. Returns the bit width of the type if `src` is zero.

### `CountLeadingZeros`
Count the number of leading zero bits.
```
CountLeadingZeros { dst: Reg, src: Reg, ty: ScalarType }
```
`dst = clz(src)`. Returns the bit width of the type if `src` is zero.

### `PopCount`
Count the number of set (1) bits.
```
PopCount { dst: Reg, src: Reg, ty: ScalarType }
```
`dst = popcount(src)`. Also known as Hamming weight.

## Control Flow

### `Branch`
Conditional branch with then/else blocks.
```
Branch { cond: Reg, then_ops: Vec<KernelOp>, else_ops: Vec<KernelOp> }
```
If `cond` is true, execute `then_ops`; otherwise execute `else_ops`.

### `Loop`
Counted loop.
```
Loop { count: Reg, iter_reg: Reg, body: Vec<KernelOp> }
```
Execute `body` `count` times. `iter_reg` holds the current iteration index (0..count-1).

### `Break`
Early exit from a loop.
```
Break
```

## Thread Indexing

### `QuarkId`
Global thread index.
```
QuarkId { dst: Reg }
```
Equivalent to `gl_GlobalInvocationID.x` (GLSL), `thread_position_in_grid` (MSL).

### `QuarkCount`
Total number of dispatched quarks.
```
QuarkCount { dst: Reg }
```

### `ProtonId`
Thread index within the proton (warp/subgroup). Quanta's terminology:
quark = thread, proton = warp, nucleus = workgroup.
```
ProtonId { dst: Reg }
```
Equivalent to `thread_index_in_simdgroup` (MSL) / `gl_SubgroupInvocationID`.

### `NucleusId`
Nucleus (workgroup) index.
```
NucleusId { dst: Reg }
```
Equivalent to `threadgroup_position_in_grid` (MSL) / `gl_WorkGroupID`.

### `ProtonSize`
Number of quarks per proton (warp/subgroup width).
```
ProtonSize { dst: Reg }
```

### `SubgroupSize`
Active subgroup size for the current dispatch.
```
SubgroupSize { dst: Reg }
```

## Synchronization

### `Barrier`
Workgroup execution barrier.
```
Barrier
```
All quarks in the workgroup must reach this point before any proceed past it.
Also acts as a memory fence for shared memory.

### `AtomicOp`
Atomic read-modify-write on a buffer field.
```
AtomicOp { dst: Reg, field: u32, index: Reg, val: Reg, op: AtomicOp, ty: ScalarType }
```

| `op` | Operation |
|------|-----------|
| `Add` | `*addr += val` |
| `Sub` | `*addr -= val` |
| `Min` | `*addr = min(*addr, val)` |
| `Max` | `*addr = max(*addr, val)` |
| `And` | `*addr &= val` |
| `Or` | `*addr \|= val` |
| `Xor` | `*addr ^= val` |
| `Exchange` | `*addr = val` |
| `CompareExchange` | CAS (see `AtomicCas`) |

All return the previous value in `dst`.

### `AtomicCas`
Atomic compare-and-swap.
```
AtomicCas { dst: Reg, field: u32, index: Reg, expected: Reg, desired: Reg, ty: ScalarType }
```
If `*addr == expected`, set `*addr = desired`. Returns old value in `dst`.

## Wave (Warp/Subgroup) Operations

### `WaveShuffle`
Exchange data between lanes within a wave.
```
WaveShuffle { dst: Reg, src: Reg, lane_delta: Reg, ty: ScalarType }
```
`dst = src` from lane `(current_lane + lane_delta)`.

### `WaveBallot`
Bit mask of which lanes satisfy a predicate.
```
WaveBallot { dst: Reg, predicate: Reg }
```
`dst` is a bitmask where bit `i` is set if lane `i` has `predicate == true`.

### `WaveAny`
True if any lane in the wave satisfies the predicate.
```
WaveAny { dst: Reg, predicate: Reg }
```

### `WaveAll`
True if all lanes in the wave satisfy the predicate.
```
WaveAll { dst: Reg, predicate: Reg }
```

### `SubgroupReduceAdd`
Sum a value across all lanes in the subgroup.
```
SubgroupReduceAdd { dst: Reg, src: Reg, ty: ScalarType }
```
`dst = sum of src across all active lanes`. Maps to `OpGroupNonUniformIAdd`
(SPIR-V), `simd_sum` (Metal), `redux.sync.add` (PTX).

### `SubgroupReduceMin`
Minimum across all lanes in the subgroup.
```
SubgroupReduceMin { dst: Reg, src: Reg, ty: ScalarType }
```
`dst = min(src) across all active lanes`.

### `SubgroupReduceMax`
Maximum across all lanes in the subgroup.
```
SubgroupReduceMax { dst: Reg, src: Reg, ty: ScalarType }
```
`dst = max(src) across all active lanes`.

### `SubgroupExclusiveAdd`
Exclusive prefix sum across subgroup lanes.
```
SubgroupExclusiveAdd { dst: Reg, src: Reg, ty: ScalarType }
```
`dst = sum of src in all lanes with index < current lane`. Lane 0 gets 0.

### `SubgroupInclusiveAdd`
Inclusive prefix sum across subgroup lanes.
```
SubgroupInclusiveAdd { dst: Reg, src: Reg, ty: ScalarType }
```
`dst = sum of src in all lanes with index <= current lane`.

## Math

### `MathCall`
Built-in math function call.
```
MathCall { dst: Reg, func: MathFn, args: Vec<Reg>, ty: ScalarType }
```

| `func` | Args | Description |
|--------|------|-------------|
| `Sin` | 1 | Sine |
| `Cos` | 1 | Cosine |
| `Tan` | 1 | Tangent |
| `Asin` | 1 | Arc sine |
| `Acos` | 1 | Arc cosine |
| `Atan` | 1 | Arc tangent |
| `Atan2` | 2 | Two-argument arc tangent |
| `Sqrt` | 1 | Square root |
| `Rsqrt` | 1 | Reciprocal square root (`1/sqrt(x)`) |
| `Exp` | 1 | `e^x` |
| `Exp2` | 1 | `2^x` |
| `Log` | 1 | Natural logarithm |
| `Log2` | 1 | Base-2 logarithm |
| `Pow` | 2 | Power (`base^exp`) |
| `Abs` | 1 | Absolute value |
| `Min` | 2 | Minimum |
| `Max` | 2 | Maximum |
| `Clamp` | 3 | Clamp to range (`min(max(x, lo), hi)`) |
| `Floor` | 1 | Floor |
| `Ceil` | 1 | Ceiling |
| `Round` | 1 | Round to nearest |
| `Fma` | 3 | Fused multiply-add (`a*b + c`) |

## Vector

### `VecConstruct`
Build a vector from scalar components.
```
VecConstruct { dst: Reg, components: Vec<Reg>, ty: ScalarType }
```
`components.len()` determines the vector width (2, 3, or 4).

### `VecExtract`
Extract a scalar component from a vector.
```
VecExtract { dst: Reg, vec: Reg, component: u8, ty: ScalarType }
```
`component`: 0=x, 1=y, 2=z, 3=w.

### `Dot`
Dot product of two vectors.
```
Dot { dst: Reg, a: Reg, b: Reg, width: u8, ty: ScalarType }
```
`dst = dot(a, b)`. `width` is the vector width (2, 3, or 4). Result is a scalar.

### `MatMul`
Matrix multiply.
```
MatMul { dst: Reg, a: Reg, b: Reg, size: u8, ty: ScalarType }
```
`size` is the matrix dimension (2, 3, or 4 for square matrices).

## Texture

### `TextureSample2D`
Sample a 2D texture at floating-point coordinates.
```
TextureSample2D { dst: Reg, texture: u32, x: Reg, y: Reg, ty: ScalarType }
```

### `TextureSample3D`
Sample a 3D texture.
```
TextureSample3D { dst: Reg, texture: u32, x: Reg, y: Reg, z: Reg, ty: ScalarType }
```

### `TextureWrite2D`
Write a value to a 2D texture at integer coordinates.
```
TextureWrite2D { texture: u32, x: Reg, y: Reg, value: Reg, ty: ScalarType }
```

### `TextureLoad2D`
Load a texel from a 2D texture at integer coordinates (no filtering).
```
TextureLoad2D { dst: Reg, texture: u32, x: Reg, y: Reg, ty: ScalarType }
```
`dst = texture[x, y]`. Unlike `TextureSample2D`, this uses integer pixel
coordinates and performs no interpolation. Maps to `OpImageRead` (SPIR-V),
`read` (Metal), `tex2Dfetch` (PTX).

### `TextureSize`
Query texture dimensions.
```
TextureSize { dst_w: Reg, dst_h: Reg, texture: u32 }
```

## Miscellaneous

### `Copy`
Register-to-register copy (for loop-carried variables).
```
Copy { dst: Reg, src: Reg, ty: ScalarType }
```

### `DeviceCall`
Call a user-defined `#[quanta::device]` function.
```
DeviceCall { dst: Reg, func_name: String, args: Vec<Reg>, ty: ScalarType }
```

### `Dispatch`
Dynamic parallelism — launch a sub-kernel from the GPU.
```
Dispatch { wave: Reg, groups: [Reg; 3] }
```
